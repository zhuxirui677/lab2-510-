# TECHIN 510 — Lab 2: Anatomy of Coding Agents

## How to Run

```bash
# Install dependencies
pip install -r requirements.txt

# Run the Tip Calculator
streamlit run app.py

# Run the GIX Eligibility Checker (Cursor version)
streamlit run eligibility_cursor.py
```

## Project Structure

- `app.py` — Tip Calculator main app
- `utils.py` — Utility functions
- `eligibility_cursor.py` — GIX Career Eligibility Checker (Component E, Cursor)
- `eligibility_chatgpt.py` — GIX Career Eligibility Checker (Component E, ChatGPT)
- `.cursorrules` — Cursor AI configuration
- `CLAUDE.md` — Claude Code project context
- `component_a_interview.md` — Staff interview notes and flowchart
- `component_b_lab.md` — Lab exercises (Levels 1–4)
- `component_c_architecture.md` — System architecture and design decisions
- `component_d_testing.md` — Regression testing and validation
- `component_e_challenge.md` — Prompt Showdown applied challenge
- `reflection.md` — Week 2 reflection
